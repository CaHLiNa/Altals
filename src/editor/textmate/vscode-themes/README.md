本目录 vendoring 自 [microsoft/vscode](https://github.com/microsoft/vscode) 的官方 theme 文件。

- 上游来源：
  - `extensions/theme-defaults/themes/2026-dark.json`
  - `extensions/theme-defaults/themes/dark_modern.json`
  - `extensions/theme-defaults/themes/dark_plus.json`
  - `extensions/theme-defaults/themes/dark_vs.json`
- 当前用途：为 LaTeX-Workshop TextMate grammar 提供与 VS Code 官方 `2026 Dark` 一致的 token color 规则
- 许可：MIT，详见 VS Code 仓库根目录 LICENSE

如需升级颜色规则，应优先同步上游 theme 文件，而不是在本仓库继续手工维护 LaTeX 专用配色。
